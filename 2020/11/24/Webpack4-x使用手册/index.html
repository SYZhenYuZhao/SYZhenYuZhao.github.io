lodaer配置

1、解析js、jsx

安装babel-loader、@babel/core 用于解析转义 .js|jsx文件

npm i babel-loader @babel/core -D

babel-loader支持多种配置方式详情见官网,本文选用 .babelrc的方式配置babel-lodaer内容如下：

安装 @babel/preset-env @babel/preset-react @babel/plugin-transform-runtime

npm i @babel/preset-env @babel/preset-react @babel/plugin-transform-runtime -D

2、解析css

安装 style-loader css-loader

npm i style-loader css-loader -D

css-loader配置cssmodule

3、解析less

安装less及less-loader，这里需要注意一下使用less一定要先安装less， 之前只做了解析忘记了安装less，结果编译时候报错，调试了半天才找到原因

npm i less less-loader -D

Plugins 配置

clean-webpack-plugin 清理dist目录

npm i clean-webpack-plugin -D

html-webpack-plugin 将打包好的js、css引入生成的模板文件中

npm i html-webpack-plugin -D

mini-css-extract-plugin 抽离文件中的css 否则css会跟js打包在一起
